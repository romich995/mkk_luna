# Тестовое заданние от ucar

## Развертывание проекта

### Копируем репозиторий
```bash
git clone https://github.com/romich995/mkk_luna.git
```

### Создаем папку data
```bash
cd ./mkk_luna
mkdir data
```

### Билдим проект
```bash
docker compose build
```

### Запуск проекта 
```bash
docker compose up
```
